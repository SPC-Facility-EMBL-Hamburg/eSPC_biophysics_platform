Overview

The binding affinity calculation can be divided into two main steps. First, we load the data, set the ligand concentration of each capillary, and the cold and hot regions. Second, we fit the Fnorm (Fhot / Fcold) versus ligand concentration curve to estimate the binding affinity. You can watch ThermoAffinity in action in the following video.

Step 1. Load data

Go to the "User guide" section, and download the file "demo.xlsx". This file contains 16 measurements of a titration curve starting at 150 μM. Then, go to the "1. Load Input" section, press the "Browse..." button, and upload the just downloaded file.

Before proceeding to the next step select 0 to 1 sec as the hot region and normalize the fluorescence signal. It has been proven that it is better to analyze the signal right after the thermal perturbation (López-Méndez, Uebel et al., 2021) .

Step 2. Fit the Fnorm versus ligand concentration curve

Go to the "2. Fitting" section, use 25 nM as the protein concentration and press the "Run Fitting" button. You should have estimated a Kd of around 800 nM.


If you want to export the information extracted from the data, go to the "3. Export results" section.
