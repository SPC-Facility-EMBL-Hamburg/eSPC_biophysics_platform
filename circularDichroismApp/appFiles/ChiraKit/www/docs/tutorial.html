1) Basic manual processing of protein and buffer spectra
  1. Download the files R78906.d01, R78906.d02, R78906.d03, R78907.d01, R78907.d02 and R78907.d03. These files contain CD measurements of a protein sample ('R78907') and the corresponding buffer ('R78906'), kindly provided by Nykola Jones from Aarhus University, Denmark.
  2. Import the six files simultaneously (drag and drop, or click 'Browse' and select them).


  3. Navigate to the 'HT signal' tab. Adjust the 'HT max value' parameter to 6.5. The wavelength range should automatically update (to 178 - 330 nm).


  4. Switch to the 'CD signal / working units' tab.
  5. In the box '4. Plotting', select only the three protein measurements.
  6. In the box '3. Processing', set the following options:
    Operation: 'Average'
    Spectrum/a 1: 'Selected in Show Column'
    Operation Units: 'Millidegrees'
    Output name: 'sample average'
  7. Click 'Apply' to execute the processing step.


  8. Repeat the previous steps, this time selecting the three buffer measurements. Name the new spectrum as 'buffer average'.
  9. Subtract the buffer baseline by using the following options in the '3. Processing' box:
    Operation: 'Subtract'
    Spectrum/a 1: 'sample average'
    Operation Units: 'Millidegrees'
    Spectrum 2: 'buffer average'
    Output name: 'baseline subtracted'


  10. Input the parameters of the 'baseline subtracted' spectrum: Set the concentration to 1.3 mg/ml, path length to 0.1 mm, number of peptide bonds to 128, and molecular weight to 14300 Da. Select 'mean unit molar ellipticity' as the working unit. If desired, delete all previous experiments to keep the workspace organised.



  11. Go to the 'Export data' panel and export the spectrum in delta epsilon units.


  12. Congratulations! You have successfully processed CD measurements, obtaining a baseline-subtracted spectrum in delta epsilon units. Be aware that the exported file contains also temperature data which was read from the original input files metadata.
2) Basic automatic processing of protein and buffer spectra
  1. Download the files R78906.d01, R78906.d02, R78906.d03, R78907.d01, R78907.d02 and R78907.d03. These files contain CD measurements of a protein sample ('R78907') and the corresponding buffer ('R78906'), kindly provided by Nykola Jones from Aarhus University, Denmark.
  2. Set the 'Input mode' to 'Automatic baseline subtraction'. Upload the 'R78907' files as 'Sample files' and the 'R78906' files as 'Baseline files'.


  3. Click on the 'Process data' button. Use 'millidegrees' as input units and 'mean unit molar extinction' as desired units.


  4. Enter the required experimental details.
    Concentration: 1.3 mg/ml
    Molecular weight: 14300
    Path length: 0.1 mm
    Number of peptide bonds: 128


  5. Press 'Submit' and wait some seconds. The finalised spectrum should appear.


  6. Adjust the 'HT max value' parameter to 6.5. The wavelength range should automatically update (to 178 - 330 nm).
  7. Go to the 'Export data' panel. Using the 'Specific CD curve(s)' option, export the 'Baseline subtracted' spectrum in delta epsilon units.
  8. Congratulations! You have successfully processed CD measurements in an automated fashion, obtaining a baseline-subtracted spectrum in delta epsilon units.
3) Analysis of protein melting curves using a single-wavelength approach
  1. Download the file thermalRampChirascan.csv, kindly provided by Osvaldo Burastero from European Molecular Biology Laboratory (EMBL), Hamburg, Germany. This file contains a thermal denaturation experiment of a protein sample.
  2. Import the file into ChiraKit. Set the wavelength range to 210 - 280 nm to remove noisy data.


  3. Navigate to the 'Thermal unfolding' panel (Section '2. Analysis'). Press on '2a. Create dataset'. Then press on '2c. Find wavelength'.
  4. Explore how the CD signal changes according to the different proposed wavelengths. Then set the 'Selected wavelength(s)' to '220-226'.


  5. Select the 'Equilibrium Two-State' unfolding model, activate the 'Fit unfolded slope' option and fit the thermal unfolding curves by clicking on '2d Fit data'.


  6. Congratulations! You have successfully fitted a reversible two-state unfolding model to obtain the melting temperature of a protein sample.

4) SVD based analysis of protein melting curves

To use the whole wavelength range information and reduce noise, we can decompose the CD signal into a set of basis spectra and their associated coefficients. Subsequently, instead of fitting the CD signal at a certain wavelength, we fit the SVD coefficients.


  1. Download the file thermalRampChirascan.csv, kindly provided by Osvaldo Burastero from European Molecular Biology Laboratory (EMBL), Hamburg, Germany. This file contains a thermal denaturation experiment of a protein sample.
  2. Import the file into ChiraKit. Set the wavelength range to 210 - 280 nm to remove noisy data.


  3. Navigate to the 'Thermal unfolding' panel (Section '2. Analysis'). Press on '2a. Create dataset'. Then change the analysis method to 'Spectra decomposition (SVD)'.



  4. Click on '2c. Decompose spectra'. Explore the tabs 'Basis spectra', 'Reconstructed spectra', 'Explained variance' and 'Coefficients' tabs.


  5. The first acquried spectrum seems to be an outlier. To remove it, right click the '1. Temperature data' Table and press 'Remove row'. Subsequently, create the dataset and find the basis spectra again.


  6. Activate the 'Fit unfolded slope' option and fit the thermal unfolding curves by clicking on '2f. Fit data'.


  7. Congratulations! You have successfully 1) applied the SVD algorithm to find a basis spectrum, and 2) fitted a reversible two-state unfolding model to obtain the melting temperature of a protein sample.


  8. Extra step: compare the obtained parameters with the ones from the Tutorial #3. Consider the robustness of each approach.
    Tip: when comparing samples, always use the same analysis method.
5) Calculation of secondary structure with Selcon3
  1. Download the file Myoglobin.csv and OpcA.csv. These files contain the spectra in delta epsilon units of Myoglobin and OpcA. The data was downloaded from the Protein Circular Dichroism Data Bank (Ramalli, Sergio Gomes et al., 2022). The CD spectrum of Myoglobin was published in Lees, Jonathan G. et al., 2006. The CD spectra of OpcA and Wheat germ agglutinin was published in Abdul-Gader, Ali et al., 2011.
  2. Import the two files simultaneously. Set the 'Input units' and the 'Working units' to mean unit molar extinction'.




  3. Navigate to the 'Secondary structure estimation' panel (Section '2. Analysis'). Adjust the 'Lower WL' parameter to 180. As a result, the AU-SMP18 reference dataset, comprising soluble and membrane proteins, will be used. Press on 'Run estimation!'.
  4. You should be able to see the percentage of each secondary structure elements and the fitted spectra.



  5. Congratulations! You have successfully estimated the percentage of secondary structure elements using the Selcon3 algorithm and the AU-SMP180 reference dataset.
  6. Extra step: Upload the PDB files 1a6m.cif (Vojtěchovský, Jaroslav et al., 1999) and 2vdf.cif (Cherezov, Vadim et al., 2008). Compare the estimation to the actual secondary structure elements.

6) Estimation of DNA-ligand binding affinities
  1. Download the file wl_385_binding_dataset.csv. These files contains the induced CD (ICD) signal for berenil (10 μM) in the presence of increasing concentrations of calf thymus DNA. The data was digitalized (thanks to WebPlotDigitizer) from Figure 1 of Garbett, Nichola C. et al., 2007.
  2. Import the file and navigate to the 'Custom analysis' panel (Section '2. Analysis'). Replace the 'Paramater name' field with 'DNA_micromolar'. Fill the second column of the Table with the following concentrations (you can copy and paste):

    0.0018

  3. Press on '2a. Create dataset'. Turn on the the 'Show plot export options' and activate the 'Log scale for the x-axis' option.


  4. Input '0.5 * (ABsignalPos - freeAsignalPos) * ((10**logTenOfKd+10+DNA_micromolar) - sqrt((10**logTenOfKd+10+DNA_micromolar)**2 - 4*10*DNA_micromolar)) + freeAsignalPos*10' as the 'Fitting function'. The parameters 'freeAsignalPos' and 'ABsignalPos' refer respectively to the baseline with no DNA and the baseline at a saturating concentration. They contain the pattern 'Pos' to force it to be positive. 10 corresponds to the compound concentration, in micromolar.
  5. Click on '2f. Get initial estimates' and subsequently on '2e. Fit data'.


  6. Inspect the fitted parameters. The Kd should be in the micromolar range (10^'logTenOfKd').


  7. Congratulations! You have successfully used a custom model to estimate the binding affinity of a DNA-ligand system.
  8. Some extra tips:
    1) If the compound concentration were 22 micromolar, the string for the fitting equation would be: '0.5 * (ABsignalPos - freeAsignalPos) * ((10**logTenOfKd+22+DNA_micromolar) - sqrt((10**logTenOfKd+22+DNA_micromolar)**2 - 4*22*DNA_micromolar)) + freeAsignalPos*22'
    2) If the baseline for the complex had negative values, then replace 'ABsignalPos' with 'ABsignalNeg'
    3) If the fitting did not work, try changing the initial values of the parameter estimates manually.
