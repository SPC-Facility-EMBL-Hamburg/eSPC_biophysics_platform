Binaries and source code downloaded from:

http://swift.cmbi.ru.nl/gv/dssp/

DSSP v2.0.4 was tested to be compatible with dssp_reform.py and dssp_traj.py

This version of DSSP was obtained under the Boost Software License verson 1 (see subdir). 
We have not modified or own the DSSP code in any way.

to use DSSP on a computer with linux operation system, you may need to give permission 
for execution (chmod +x <filename> ) to one of the three binary files found in this directory.

Use the win32 version on Windows operating systems, or one of the linux binaries
approriate for your architecture. Alternatively you can compile a binary from the
source code (in subdir dssp-2.0.4). See the website for further notes on that.

Please be aware that this version of DSSP is not processing trajectories by default.
The standard input files for DSSP are three-dimensional protein models in a protein
databank (PDB) format.
