Overview

The binding affinity calculation can be divided into three main steps. First, we load the data, set the ligand concentration of each capillary/well, and the temperature range of interest. Second, we fit each melting curve to obtain the unfolded fraction versus temperature for each ligand concentration. Third, we compute the binding affinity at any fixed temperature from the unfolded fraction versus ligand concentration curve. You can watch FoldAffinity in action in the following video

Step 1. Load data

Go to the "User guide" section, and download the file "nDSFdemoFile.xlsx" (Input file section). This file contains 16 measurements (by duplicate) of a titration curve starting at 5000 Î¼M. The last 6 capillaries should be discarded. Then, go to the "1. Load Input" section, press the "Browse..." button, and upload the just downloaded file.

Select the "Autofill Table" option, set 5000 as the initial ligand concentration, and then unselect the "Autofill Table" so you can later edit manually the "Position versus Concentration" Table. Unselect the last six capillaries from the "Position versus Concentration" Table and change the units of "Ligand Concentration" to micromolar. Select 'Ratio' as the fluorescence signal.

Before proceeding to the next step select 45 and 90 degrees as the lower and upper temperature limits. It is important for the fitting routine that the temperature range covers no more than 40-50 degrees.

Step 2. Fit the fluorescence curves

Go to the "2. Fit fluorescence" section and fit the melting curves with the "Global_CP" fitting routine by clicking on the "Run Fitting" button. Now we have found for each ligand concentration (capillary/well), a function that has as input the temperature and output the unfolded fraction.

Step 3. Fit the unfolded fraction

Go to the "3. Fit unfolded fraction" section, set 9.9 as the protein concentration, use 59-65 as the Isothermal Fit range, and press on "Run Fitting" to obtain the binding affinity estimates.


If you want to export the information extracted from the data, go to the "4. Export results" section.
