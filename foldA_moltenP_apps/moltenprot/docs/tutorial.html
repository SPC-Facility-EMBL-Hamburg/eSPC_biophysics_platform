Overview

The protein stability analysis can be divided into three steps. First, we load the data, set the conditions names of each capillary/well, the signal to use, and the temperature range of interest. Second, we fit each melting curve according to an unfolding model. Third, we analyze the information derived from the models such as the estimated melting temperatures and the fraction unfolded versus temperature curves. You can watch MoltenProt in action in the following video

Step 1. Load data

Go to the "User guide" section, and download the file "nDSFdemoFile.xlsx" (Input file section). This file contains 48 measurements of a protein under different conditions with Urea and Guanidinium chloride. Then, go to the "1. Load Input" section, press the "Browse..." button, upload the just downloaded file. The conditions names will be loaded from the xlsx file ('Overview' sheet).


Try the different signal normalization options, the median filter and observe its effect on the melting curves. For this tutorial we will only analyze the conditions with Urea. We have two options, deselecting the Include option in the conditions with Guanidinium chloride, or changing the Series from A to B in the column "Series" of the conditions with Urea and then selecting the Series B in the "Select Series" button.

Step 2. Fitting

Go to the "2. Fitting" section and fit the melting curves with the Equilibrium Two State model. Now we have estimated for each condition (capillary/well), the enthalpy of unfolding and the melting temperature ( Tm ). Sort the data by the estimated Tm and notice the correlation with the Urea concentration.

Step 3. Analysis

Go to the "3. Analyze" section, and explore the different results plot available. With the Equilibrium Two state model there are three options: Tm versus condition, Tm versus Tonset (1 % unfolded), and fraction unfolded versus temperature.


If you want to export the information extracted from the data, go to the "4. Export results" section. Try the Generate Report option.
